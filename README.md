# Authentication api

A Json-Web-Token based rest api for authentication built on top of Expressjs + MongoDB.

## Setup
**You will need :**
- Nodejs
- MongoDB
- Cloudinary api access


## Installation
> Sets the env variables in .env.dev and .env.test according to .env.sample's content

on command-line run :

```bash
npm install
```

## Usage

### For testing

```bash
npm test
```

### For development

```bash
npm run dev
```
